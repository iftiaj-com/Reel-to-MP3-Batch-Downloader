# Reel to MP3 Batch Downloader

A modern, mobile-friendly Streamlit web application that downloads Instagram Reels as MP3 audio. 
It supports pasting unstructured text (like WhatsApp chat copies) to extract and convert Reels.

## Running Locally

1. Install requirements:
   ```bash
   pip install -r requirements.txt
   ```
2. Launch Streamlit:
   ```bash
   streamlit run app.py
   ```
